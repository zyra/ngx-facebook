Save Button (class exported as FBSaveComponent)

The Save button lets people save items or services to a private list on Facebook, share it with friends, and receive relevant notifications.

Official Facebook documentation for this class can be found here.

Usage 
<fb-save uri="https://github.com/zyra/ng2-facebook-sdk/"></fb-save>
Inputs uri (string)

The absolute link of the page that will be saved. Current Link/Address
